# personal-portfolio
A modern personal portfolio website built using HTML and CSS featuring glassmorphism design, smooth animations and responsive layout.

## 🚀 Features

- Responsive portfolio layout
- Modern navy blue and dark pink design
- Glassmorphism UI effect
- Smooth fade animations
- Skill progress bars
- Hover effects and modern buttons
- Clean and simple user interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Responsive Web Design
- CSS Animations
- Glassmorphism Design
